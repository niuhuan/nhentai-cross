一个美观且跨平台的*NHentai客户端*，支持桌面端与移动端(Mac/Windows/Linux/Android/IOS)。

内置DNS拦截器，可以让部分国家和地区免代理使用网络加速，请您在遵守当地法律的情况下使用。(需要在设置中开启）

如果您觉得此软件对您有帮助，可以star进行支持。同时欢迎您issue，一起让软件变得更好。
